# Sitecore Marketplace Resources

## AI-Powered Apps
- [Marketers Chat](https://github.com/izharikov/marketers-chat) - AI-powered chat interface for content marketers to manage SitecoreAI resources through natural language.
- [SEO Schema](https://github.com/izharikov/seo-schema-marketplace-app) - generates Schema.org JSON-LD structured data for SitecoreAI pages to improve SEO rich results.
- [LLMify](https://github.com/Sitecore-Hackathon/2026-NaN) - extension that leverages artificial intelligence to automatically convert Sitecore pages' HTML content into clean, structured Markdown, and to generate LLMs.txt file.
