## AI-Powered Apps

- [Marketers Chat](https://github.com/izharikov/marketers-chat) - AI-powered chat interface for content marketers to manage Sitecore AI resources through natural language.
- [SEO Schema](https://github.com/izharikov/seo-schema-marketplace-app) - Generates Schema.org JSON-LD structured data for Sitecore AI pages to improve SEO rich results.
- [LLMify](https://github.com/Sitecore-Hackathon/2026-NaN) - Leverages AI to automatically convert Sitecore pages' HTML content into clean, structured Markdown and generate LLMs.txt files.

## AI Integrations

- [Vercel AI SDK](https://ai-sdk.dev/docs/introduction) - Framework for building AI-powered apps with streaming support.
- [AI Elements by Vercel](https://ai-sdk.dev/elements) - Pre-built UI components for AI chat interfaces.
- [sitecore-ai-sdk-tools](https://github.com/izharikov/sitecore-ai-sdk-tools) - AI SDK tools for interacting with Sitecore Marketplace APIs.

## Documentation

- [Getting Started](https://doc.sitecore.com/mp/en/developers/sdk/latest/sitecore-marketplace-sdk/quick-start--cli-.html) - CLI quickstart using `npx shadcn@latest add` to scaffold your environment.
- [Sitecore Marketplace Developer Basics](https://learning.sitecore.com/partners/learn/courses/1502/sitecore-marketplace-developer-basics) - Official e-learning course covering marketplace fundamentals.
- [marketplace-sdk](https://github.com/Sitecore/marketplace-sdk) - Official Marketplace SDK source code.
- [Blok](https://github.com/sitecore/blok) - Sitecore's product design system.
- [Apps Architecture](https://doc.sitecore.com/mp/en/developers/marketplace/app-architecture-and-authorization-options.html) - Architecture and authorization options for marketplace apps.

## Example Apps

- [Google Analytics App](https://github.com/Sitecore/marketplace-google-analytics) - Official example integrating Google Analytics with Sitecore.
- [Icon Picker](https://github.com/Sitecore/marketplace-icon-picker) - Official example app for picking icons within the marketplace.
- [Hackerspace Workshop](https://github.com/Sitecore/hackerspace-workshop) - Hands-on workshop from Sitecore Symposium 2025.
- [Marketplace Starter](https://github.com/izharikov/marketplace-starter) - Minimal starter template for a marketplace app.
- [Full-Stack Custom Auth](https://github.com/izharikov/app-full-stack-custom-auth) - Full-stack example with custom authentication flow.
