# Contributing

Thank you for your interest in contributing to this list! Here are a few guidelines.

## Adding a Resource

1. Fork this repository.
2. Add your resource to the appropriate section in `README.md`.
3. Use the format: `- [Name](url) - Short description ending with a period.`
4. Keep descriptions concise (one sentence).
5. Check that the link is working and points to a quality resource.
6. Submit a pull request with a brief explanation of why the resource is useful.

## Quality Standards

- The resource should be directly related to Sitecore Marketplace development.
- Avoid duplicates — search existing entries before adding.
- Open-source projects should have a README, license, and some level of documentation.
- Blog posts and tutorials should be up-to-date and provide practical value.

## New Sections

If your resource doesn't fit an existing section, propose a new one in your pull request description.
